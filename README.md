# RGB LED Control System for STM32F4

Проект системы управления RGB LED-лентами для комнаты на базе микроконтроллера **STM32F4**.  
Основная идея — создать модульную и расширяемую систему освещения с физическими кнопками, сценариями и возможностью дальнейшего развития (реле, датчики, интерфейсы связи).

## Цели проекта

- Управление RGB LED-лентами в комнате
- Питание контроллера от общего 12 В блока питания LED-лент
- Управление яркостью и цветом (PWM)
- Реализация логики режимов освещения
- Работа с физическими кнопками
- Учебно-практический проект для STM32

## Архитектура (план)

### Питание
- Входное питание: **12 В**
- Понижающий преобразователь: **12 В → 5 В (≈3 А)**
- Питание STM32 от 5 В (через стабилизатор/USB/VIN)

### Управление LED
- RGB / RGBW LED-ленты на 12 В
- Управление каналами через:
  - MOSFET 
  - PWM с таймеров STM32

### Управление пользователем
- Физические кнопки 
- Энкодер с кнопкой
- Возможность реализации логики:
  - Вкл / выкл
  - Переключение режимов
  - Изменение яркости
  - Сценарии освещения

## Логика управления (первый этап)

На первом этапе планируется:
- Простая логика включения и выключения
- Несколько режимов подсветки
- Управление кнопками без меню и дисплея
- Максимально понятная и надежная схема

В дальнейшем:
- Сценарии освещения
- Плавные переходы
- Расширение количества зон
- Подключение реле и датчиков

## Используемый микроконтроллер

- **STM32F4 series**
- LL
- PWM через аппаратные таймеры
- Возможность отладки через UART
